# Sew_nsLay
Sew_nsLay adalah platform berbagi inspirasi fashion yang terinspirasi dari konsep Pinterest, namun fokus pada pola jahitan, desain pakaian, dan referensi fashion. Pengguna dapat mengunggah desain, melihat karya orang lain, serta menemukan inspirasi untuk proyek jahit-menjahit atau fashion mereka.

Fitur Utama
Upload Pola & Desain : Pengguna dapat mengunggah gambar desain, pola jahitan, atau referensi fashion
Kategori Desain : Desain dapat dikelompokkan berdasarkan kategori (misalnya: gaun, outer, rok, hijab, kidswear)
Like & Favorite : Pengguna bisa menyukai atau menyimpan desain favorit
Exploration Page : Menjelajah konten layaknya Pinterest versi fashion
